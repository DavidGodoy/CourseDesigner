# CourseDesigner
Diseñador visual de cursos basado en Eclipse y el desarrollo dirigido por modelos

Necesita el Kepler Modeling Tools Service Release 2 (probado en build: 20140224-0627) junto a otros componentes:

-GMF Tools
-EMFStore SDK
-OCL Examples and Editors SDK
-ECP SDK 3.x
-MOFScript 1.3.3 o superior. Los plugins son: model, parser, runtime, editor, traceability, fileresourcemodel (directamente a la carpeta plugins de Eclipse)
-jdom.jar (directamente a la carpeta plugins de Eclipse
